filelist
========
Temporary panel, virtual folder that allows keeping links to frequently used files.

## Compatibility
- `doublecmd-gtk2`
- `doublecmd-qt4`
- `doublecmd-qt5`

## Notes
A list of files is loaded from `$DC_CONFIG_PATH/filelist.txt`.
