vte_ncdu
========
Embeds Virtual Terminal Emulator (VTE) widget (GTK2 version) and runs `ncdu`.

![plug-preview](https://i.imgur.com/fiWfyAE.png)

## Compatibility
- `doublecmd-gtk2`
