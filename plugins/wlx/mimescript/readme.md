mimescript
========
Displays various information about file using command line utilities. Detection by MIME type.

![plug-preview](https://i.imgur.com/OqeNQE5.png)

## Compatibility
- `doublecmd-gtk2`

## Notes
Requires `gtksourceview-2.0`.
